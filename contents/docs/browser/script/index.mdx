<StepItem title="Usage">

<Note title="Important" type="danger" className="mb-8">
Do not import `click-to-component-browser` in product enviroument.
</Note>

Import `click-to-component-browser` in your project's entry file.

```js
if (process.env.NODE_ENV === 'development') {
  import("click-to-component-browser");
}
```
